### cmp(bytes,bytes)

```solidity

    internal
    pure
    returns (int256 result)
```

Returns 0 if `a == b`, -1 if `a < b`, +1 if `a > b`.   
If `a` == b[:a.length]`, and `a.length < b.length`, returns -1.

